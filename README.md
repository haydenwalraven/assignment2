# Hayden's Food Finder Website

## Website Structure
- index.html (Home Page)
- restaurants.html (Restaurant Listings)
- recommend.html (Recommendation Page)
- register.html (Registration Page)
- reservation.html (Reservation Page)
- style.css (Website Styling)
- script.js (JavaScript Logic)

## JavaScript Validation Logic
- Username validation checks minimum length and valid characters.
- Email validation checks correct email format.
- Phone number validation checks number length.
- Password validation checks uppercase, lowercase, number, special character, and minimum length.
- Reservation validation prevents invalid dates and incorrect people counts.
- Deposit amount updates automatically depending on restaurant selection.
- Payment method dynamically displays voucher or credit card fields.
- Restaurant recommendation logic suggests restaurants based on dietary preference, budget, and dining purpose.
